Etiquetas Expo

  • TITLE: Esta etiqueta es usada para dar Título/Nombre a cualquier página web.

  • RUBY: La etiqueta Ruby especifica un rubí anotación, Ruby anotaciones son utilizadas para Asia Oriental tipografía, para mostrar la pronunciación de caracteres de Asia Oriental o en este caso el significado de algo.

  • RB Y RT: RB: Vendría siendo la palabra a la que se le va a dar un significado

  • RT: Vendría siendo el significado de la palabra, en esté caso

  • BDO: El elemento HTML bdo permite a los autores anular el algoritmo bidireccional para fragmentos específicos de texto. Lo que básicamente hace es especificar la dirección del texto en su contenido mediante el atributo "dir“.

  • SUB: El elemento HTML sub hace que su contenido sea representado como subíndice. Los subíndices son usualmente representados con un tamaño de fuente menor y un poco más abajo que el texto normal. Puede ser útil en algunas ciencias como, por ejemplo, la química.

  • CENTER: Esta etiqueta HTML funciona para poder centrar cualquier texto dentro de una página web
